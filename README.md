## Sales Data Analysis
 ## Objective

To analyze sales data using Python and Pandas for finding:

Total sales by product

Total sales by region

Monthly sales trends

## Tools Used

Python 3

Pandas

Matplotlib

VS Code / IDLE / Jupyter Notebook

## Files

sales_data.csv → Sales dataset

sales_analysis.py → Python analysis code

README.md → Project information

## Steps to Run

1.Install Libraries

2.pip install pandas matplotlib openpyxl


## Keep Files Together

project_folder/
├── sales_analysis.py
├── sales_data.csv


## Run the Code

In IDLE: Press F5

In VS Code: Run

python sales_analysis.py

##  What the Code Does

Loads and reads the sales CSV file.

Shows basic information and missing data.

## Calculates:

1.Total sales by product

2.Total sales by region

3.Monthly sales totals

4.Displays charts for better understanding.

## Output

You’ll see:

Tables showing sales summaries

Bar charts and line charts for sales analysis
